基本指令架構

指令

git init

初始化 Git
新增一個本地數據庫

git –verson

查看版本

設定個人資料

輸入姓名：git config –global user.name “gon”
輸入個人的 email：git config –global user.email “gonsakon@gmail.com“

git config –list

查詢 git 設定內容

(git 和 github 註冊的帳號不一定要一樣)
item2 (foe Mac 的終端機)

git status

看資料夾狀況

git add .

從工作目錄將檔案加入到索引
. 代表所有變更的檔案
或是你可以在後面寫你想加入索引的檔案名

git commit -m “first upload”

將索引區的檔案放到本地數據庫

git log

看 commit 的歷史紀錄
脫離視窗按小寫 q

推到遠端數據庫 (GitHub)

再見了 GitHub 的資料庫之後，可以在下面選擇相對應的

可以打開本地端的 config 檔看看檔案設定狀況

origin/master

是遠端數據庫的位置
(origin 在這裡是 GitHub 的角色)
