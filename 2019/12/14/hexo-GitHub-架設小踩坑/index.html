hexo + GitHub 架設小踩坑 - 主題無法顯示於 GitHub 上

Yun's Blog
2019-12-14
web前端
GitHub, Hexo

因為搜尋就有很多文章教怎麼將 Hexo 部屬在 GitHub 上，所以這篇文就不打算一個一個步驟教了，有需要的人可以滑到最下面的參考資料，兩篇文章都教得非常清楚~

但，就算教得再清楚，踩坑王還是會踩坑。問題就出在於：

當時我已經確認在 http://localhost:4000/ 是可以出現網頁的

萬分開心地選了主題後(可以到 Hexo 的主題網頁 挑選自己喜歡的主題)

發現自己部屬到 GitHub 上是長這個樣子

這時要回到
_config.yml 的 url 屬性 和 root 屬性

url: 你的網站
root: 你的專案
寫起來就會像這樣

之後要回到你的資料夾刪掉 .deploy_git 的資料夾
之後重新按照以下指令部屬：

$ hexo clean $ hexo g $ hexo d

然後他就會重新再出現一個 .deploy_git 資料夾
你的網站也會套用好你的主題了！
